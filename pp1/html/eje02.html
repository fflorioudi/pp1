HTML es el lenguaje con el que se escribe el contenido de las paginas web
Las paginas web pueden ser vistas por el usuario mediante un tipo de aplicacion llamada cliente web o mas comunmente "navegador".
Podemos decir por lo tanto que el HTML es el lenguaje usado para especificar el contenido que los navegadores deben representar a la hora de mostrar una pagina web.
